Core Features

Everything you need for autonomous browser automation powered by dual-brain AI.

🧠

Dual-Brain LLM Architecture

Planner brain for high-level strategy, Navigator brain for step-by-step execution. Two LLMs working together for autonomous task completion.

👁️

DOM Vision Engine

Real-time scanning and indexing of interactive page elements. Visual element recognition and intelligent element targeting.

🔄

Autonomous Navigation

Multi-tab orchestration, step-by-step self-healing execution, auto-retry logic when actions fail.

🗃️

Memory & State Management

Persistent session storage, agentMemory for data extraction across steps, extension state synchronization.

📲

WhatsApp Remote Control

Remote task control via WhatsApp through NanoBridge companion server. Send tasks from your phone, get results back.

🔗

Three-System Integration

Full pipeline: Open WebUI / Demo UI → VS Code Copilot Agent Mode → NanoAgent Chrome Extension. Automated end-to-end browser task execution.

🌐

NanoBridge Server

Express.js companion server with API endpoints for task submission, polling, and result retrieval. WhatsApp Web.js integration.

🔐

Privacy-First Local Processing

All AI processing happens locally. No cloud data exfiltration. Local AI model caching and on-device data processing.

🛡️

Security & Trust Infrastructure

Certificate revocation validation, Safe Browsing protection, subresource filtering.

💻

VS Code Copilot Bridge

VS Code extension that auto-forwards prompts from the Demo UI to GitHub Copilot Agent Mode. File watcher based with status bar integration.
